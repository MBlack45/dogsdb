Інструкція з запуску проекту

Цей README файл містить інструкції щодо запуску проекту, який використовує MySQL базу даних та потребує авторизації як користувач з повними правами доступу до баз даних.
Передумови

Перед початком, впевніться, що ви маєте наступні компоненти встановлені на своєму комп'ютері:

    Java Development Kit (JDK)
    MySQL сервер

Кроки для запуску проекту

1.Завантажте та встановіть Java Development Kit (JDK):

    Відвідайте офіційний веб-сайт Oracle Java та завантажте та встановіть JDK відповідно до вашої операційної системи.

2. Завантажте та встановіть MySQL сервер:

    Відвідайте офіційний веб-сайт MySQL та завантажте та встановіть MySQL сервер відповідно до вашої операційної системи. Дотримуйтеся інструкцій для налаштування користувача з повними правами доступу до баз даних.

3. Запустіть проект:

Після успішної компіляції запустіть проект, використовуючи наступну команду в терміналі, викликаному у директорії з JAR файлом проекту:

java -jar DogsDB-1.0-SNAPSHOT-jar-with-dependencies.jar

4. Авторизуйтеся в системі:

Після запуску проекту вам буде запропоновано ввести свої облікові дані для авторизації. Введіть коректні дані користувача MySQL з повними правами доступу до баз даних.

Після авторизації проект повинен успішно підключитися до MySQL бази даних та виконати відповідні операції зі створення та заповнення бази даних.
